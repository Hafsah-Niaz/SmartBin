## ğŸŒ· Project Overview

The **SmartBin IoT Monitoring System** is an intelligent waste management web app designed to monitor the **garbage levels** of multiple bins in real time.  
Each bin is equipped with a **sensor** that measures its fill level and sends data to **Firebase**, which powers the **live monitoring dashboard**.

The system ensures timely garbage collection, prevents overflow, and enhances urban waste management through automation and real-time data visualization.
